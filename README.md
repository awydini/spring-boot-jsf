# Cif Application

simple project to illustrate customer information file web application, using  spring boot, spring security, hibernate, prime faces and h2 db.

##  Project Requirements

 - maven 3.5 +
 - Jdk 1.8
 - spring boot 2.0.0
 - prime face 6.2
 - h2 db
 - junit 5
 - mockito

## How to build

    mvn clean install

## How to run 

	mvn spring-boot:run

## Execute tests

	mvn test
